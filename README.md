# Gourmet Palace

Ini merupakan desain pertama dari **Project Gourmet Palace** yang dikembangkan menggunakan **HTML**.  
Tujuan dari project ini adalah membuat tampilan awal aplikasi restoran sederhana sebagai dasar pengembangan lebih lanjut.

## 📂 Struktur Project
- Folder `Restaurant App` berisi file utama HTML.
- File tambahan akan dikembangkan seiring berjalannya project.

## 🚀 Status
Versi awal (prototype) – fokus pada desain dasar dan struktur halaman.

## 📝 Catatan
Desain Project ini masih dalam tahap awal, sehingga fitur dan tampilan merupakan konsep aplikasi Gourmet Palace pertama.
